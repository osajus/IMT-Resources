# What is this?
This is a collection of various forms and resources used by Incident Management Teams.

This program is designed to assist an incident manager in searching for relevant documents that are frequently used while involved in an incident.

# How do I use this?
This is designed to be downloaded and used on your local machine. 

Latest release (*2026-07-21-beta*) can be found **[here](https://github.com/osajus/IMT-Resources/releases/download/2026-07-21-beta/release.zip)**.


*Verification Hash: a075ff9947f819c6ccb048f3524122cb7858488555e0bcee9dc079323d2ab1e9*

Unzip the contents and run the "IMT-Resources.html" file.
